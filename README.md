#🏛️ WarOfConquerors 

> **Civilization VI × Colonist.io** — казуальная 4X пошаговая стратегия для мобильных устройств.

Стройте империи, исследуйте технологии, воюйте и ведите дипломатию с друзьями — в сессиях, созданных для реальной мобильной игры. 5 минут на быструю схватку, 90 на полноценную кампанию.

---

## 🎮 Что это такое?

WarOfConquerors  — мобильная стратегия на Unity, объединяющая глубину **Civilization VI** (гексагональная сетка, деревья технологий, 4X-механики, уникальные цивилизации) с социальным мультиплеером **Colonist.io** (лобби по коду/ссылке, одновременные ходы, асинхронная игра в облаке).

**10 цивилизаций · 5 типов победы · Онлайн и асинхронный мультиплеер · Free-to-play, нулевой pay-to-win**

## ⚙️ Технологический стек

| Уровень | Технология |
|---|---|
| Движок | Unity 2022.3 LTS (URP) |
| Язык | C# |
| Нетворкинг | Photon Fusion 2 |
| Бэкенд | Firebase + PlayFab |
| CI/CD | Unity Cloud Build + GitHub Actions |
