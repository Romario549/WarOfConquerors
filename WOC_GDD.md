## 1. Концепция

Мобильная пошаговая стратегия, объединяющая глубину 4X-формулы Civilization VI с удобным социальным мультиплеером Colonist.io. Игроки строят империи, исследуют технологии, ведут войны и дипломатию — в сессиях, подходящих для мобайла: 5–15 мин (быстро) до 30–90 мин (стандарт).

Пояснения 4Х -- 
Исследование (Explore): Разведка карты, поиск ресурсов и новых территорий.
Расширение (Expand): Основание новых городов, колоний и увеличение влияния.
Эксплуатация (Exploit): Эффективное использование ресурсов, развитие технологий и экономики.
Уничтожение (Exterminate): Устранение конкурентов военным, дипломатическим или культурным путем.

**Основной цикл:** Основать город → Собирать ресурсы → Строить районы → Исследовать технологии → Обучать юниты → Расширяться/Дипломатия → Условие победы

**Ключевые принципы:**
- **Доступная глубина** — сложные системы, интуитивность интерфейса
- **Кооп пушка** — мультиплеер основной; асинхронные ходы, лобби для друзей
- **Вариативность решения** — каждый ход содержит интересный выбор
- **Живой и развивающийся** — сезоны, новые цивилизации, события после запуска

## 5. Ресурсы и города

**8 типов ресурсов:** 
Еда (рост) · 
Производство (строительство) · 
Золото (торговля) · 
Наука (технологии) · 
Культура (инновации) · 
Жильё (лимит населения)

**Развитие города:**
- Основывается Поселенцем; минимум 3 тайла между городами
- Население растёт с излишком Еды; каждая единица населения открывает новый тайл
- **6 типов районов:** 
Кампус (Наука) · 
Торговый центр (Золото) · 
Промышленная зона (Произв.) · 
Театральная площадь (Культура) · 
Укрепление (Военное) · 
- Каждый район вмещает до 2 зданий
- **Чудеса света:** Глобальные; кто первый завершил — победил; уникальный мощный бонус

**Визуальный рост города:** Деревня → Городок → Город → Мегаполис (4 стадии, видны на карте)

## 6. Деревья технологий и инноваций

**25 технологий** в 4 эпохах (Древняя → Средние века → Возрождение → Индустриальная)
**20 гражданских инноваций** в параллельном дереве, открывающем типы правительств и карточки политик

**Правительства:** Вождество / Монархия / Республика / Демократия
**Карточки политик:** Выберите 3 из 12 доступных на правительство (упрощённая система)

**Пример пути технологий:**
```
Горное дело → Обработка бронзы → Обработка железа → Сталь
Письменность → Философия → Образование → Научная теория → [Победа науки]
Мореплавание → Кораблестроение → Навигация
```
нужно будет потом детализировать****

## 7. Бой и армия

**Классы юнитов:** Пехота · Дальнобойные · Кавалерия · Осадные · Морские (классические счётчики)
**Формула боя:** `Урон = 30 × e^((БС_атак − БС_защ) / 25)` с отклонением ±10% (формула из цивы с поправкойй на тайл на котором стоят воины)
**Продвижения:** Юниты получают опыт; на 3 порогах выбор из 2 улучшений (бинарный выбор)
**Защита города:** Стены + гарнизон + местность = Значение защиты. Осадные юниты ее снижают. При 0 защиты вражеские юниты могут зайти и забрать город себе.

## 9. Мультиплеер 

- **Лобби:** Публичная или приватная комната (код/ссылка); быстрый матч по ELO; режим наблюдателя
- **Асинхронная игра:** Пуш-уведомление в ваш ход; таймер 24ч (настраивается); полное облачное сохранение
- **Таймер хода:** Быстро = 45 сек · Стандарт = 90 сек · Асинхронно = 24 ч
- **Выходы:** ИИ берёт управление; повторное подключение разрешено; миграция хоста; голосование за досрочный конец

## 11. Арт и аудио

**Арт-стиль:** Стилизованный мультяшный 2.5D. Жирные контуры, высокий контраст, направление Civ VI для 6" экранов.
на первое время для текстур берем стиль PSX как на 1 playstation, но при этом компенсируем плавностью анимаций и шейдерами.
- Юниты: анимированные круглые жетоны на тайлах
- Лидеры: 2D анимированные портреты, 3 эмоциональных состояния (Нейтральное / Дружелюбное / Враждебное)
- UI: у каждой цивилизации уникальный основной цвет

**Аудио:**
- Адаптивный саундтрек по эпохам (этнический → оркестровый → полифонический → индустриальный) (есть товарищ музыкант, за 0 рублей сделает)
- У каждой цивилизации уникальная музыкальная тема (есть товарищ музыкант, за 0 рублей сделает)
- 3–5 голосовых реплик на лидера (приветствие / похвала / предупреждение / требование / прощание)(найдем на стоках, на крайний сами запишем)

## 12. Технический стек

| Компонент | Технология |
|---|---|
| Движок | Unity 2022.3 LTS (URP) |
| Язык | C# |
| Нетворкинг | Photon Fusion 2 |
| Бэкенд | Firebase (аутентификация/хранилище/FCM) + PlayFab (данные игроков/таблицы лидеров) |
| CI/CD | Unity Cloud Build + GitHub Actions |
| ИИ | Кастомное дерево поведения (5 уровней сложности) |

**Целевая производительность:** Средний Android 2021+ → 30 FPS / 512 МБ RAM. Топовый / iOS → 60 FPS / 1 ГБ RAM. Максимальный размер сборки: 200 МБ.
**Архитектура:** Вся игровая логика авторизована на сервере. Состояние хода сериализуется как JSON (~5 КБ/дельта). Асинхронные матчи полностью хранятся в облаке, уведомления через FCM.
