## Фаза 1 — Предпродакшн (Месяцы 1–2)

**Цель:** Зафиксировать дизайн, проверить технологии, выровнять команду. Готового продукта пока нет.

### Веха 1.1 — Настройка проекта 
- [ ] Проект Unity 2022.3 LTS инициализирован, URP настроен
- [ ] Git-репозиторий создан (монорепо: движок + бэкенд)
- [ ] CI-пайплайн GitHub Actions работает (сборка + юнит-тесты на PR)
- [ ] Цели Unity Cloud Build настроены (iOS + Android)
- [ ] Проектная доска Jira/Linear создана; определён ритм спринтов (2-недельные спринты)
- [ ] Все члены команды онбордированы; NDA подписаны; лицензии на инструменты получены
- [ ] Firebase-проект создан (Auth, Firestore, FCM, Analytics)
- [ ] Аккаунт PlayFab и тайтл настроены
- [ ] Лицензия Photon Fusion 2 получена; SDK интегрирован

### Веха 1.2 — Технический прототип 
- [ ] **Система гексагональной сетки** реализована: генерация тайлов, система координат (аксиальная), поиск соседей
- [ ] Процедурный генератор карт: назначение местности по шуму + правилам биомов
- [ ] Система тумана войны: маска видимости, состояния explored/unexplored
- [ ] Базовая камера: щипок для зума (стратегический/город/тайл), перетаскивание, ограничение краёв
- [ ] Машина состояний ходов: счётчик хода, порядок игроков, триггер конца хода
- [ ] Сериализация игрового состояния: полное состояние в/из JSON (основа для асинхронности)
- [ ] Тест производительности: 120-гексагональная карта рендерится на 30+ FPS на Samsung Galaxy A52

### Веха 1.3 — Арт и дизайн-прототипы 
- [ ] Прототип арт-стиля: 3 типа местности (Луга, Холмы, Пустыня) в целевом качестве
- [ ] Один жетон юнита (Воин): анимация покоя, подсветка выбора, твин движения
- [ ] Один визуал города: стадия Деревни, анимация пульсации строительства
- [ ] Вайрфреймы UI завершены для всех 8 ключевых экранов (Карта мира, Город, Технологии, Инновации, Дипломатия, Меню юнита, Лидер, Победа)
- [ ] UX-схема потока: полный путь клика от главного меню до первого условия победы
- [ ] Цветовая палитра и дизайн-система задокументированы (hex-коды, шрифты, правила отступов)
- [ ] Документ аудио-направления: темы эпох, руководство по SFX, список голосовых референсов

### Веха 1.4 — Фиксация дизайна 
- [ ] GDD v2.0 утверждён всеми лидами(2 шалопаями)(дальнейшие изменения объёма требуют формального запроса на изменение, у меня спросить надо)
- [ ] Все 10 карточек данных цивилизаций финализированы (характеристики, способности, счётчики проверены на бумаге)
- [ ] Дерево технологий (25 узлов) сбалансировано в таблице; время перехода между эпохами подтверждено
- [ ] Дерево инноваций (20 узлов) сбалансировано; взаимодействия правительства/политик отображены
- [ ] Все 6 условий победы определены с точными условиями триггера
- [ ] Формула боя проверена в таблице по всем сочетаниям юнитов
- [ ] Стратегия ценообразования монетизации финализирована; юридическая проверка F2P-соответствия по регионам
- [ ] Реестр рисков создан; 7 лучших рисков назначены владельцам митигации

**Критерии выхода из Фазы 1:** Гексагональная сетка работает на устройстве, арт-стиль одобрен директором, GDD зафиксирован, CI-пайплайн зелёный.

## Фаза 2 — Альфа (Месяцы 3–6)

**Цель:** Играбельный сквозной игровой цикл для внутреннего тестирования. Грубый, но функциональный.

### Веха 2.1 — Основные системы (Месяц 3)

**Гекс и карта**
- [ ] Полный рендеринг типов местности (все 8 типов + варианты)
- [ ] Система размещения ресурсов: Предметы роскоши, Стратегические, Природные чудеса
- [ ] Выбор размера карты: генерация Крошечной / Малой / Стандартной / Большой
- [ ] Панель инспектора тайла: нажатие на тайл показывает тип местности + выход + ресурс

**Города**
- [ ] Юнит Поселенца: появляется на стартовой позиции цивилизации, основывает город действием
- [ ] Владение тайлами города: визуализация радиуса захвата
- [ ] Счётчик населения: накопление излишка Еды → триггер роста
- [ ] Отображение выхода тайла: рабочие тайлы выделены; нерабочие приглушены
- [ ] Ограничения Жилья и Удобств реализованы и протестированы

**Ресурсы**
- [ ] Панель ресурсов (нижняя строка): Еда, Производство, Золото, Наука, Культура, Вера, Удобства, Жильё
- [ ] Расчёт выхода за ход: базовая местность + улучшения + здания + районы
- [ ] Золотая казна: накопление, трата, предупреждение о дефиците

### Веха 2.2 — Юниты и бой (Месяц 3–4)

**Юниты**
- [ ] Система данных юнитов: ScriptableObject на тип юнита (сила, движение, класс, стоимость)
- [ ] Спавн юнита: очередь производства города → юнит появляется на тайле города
- [ ] Система движения: поиск пути A* на гексагональной сетке; вычет очков движения; подсветка конца движения
- [ ] Правило стакинга юнитов: один военный юнит + один гражданский на тайл
- [ ] Радиальное меню действий: нажатие на юнит → Переместить / Атаковать / Укрепиться / Пропустить / Распустить

**Бой**
- [ ] Расчёт боевой силы: базовая + модификатор местности + бонус укрепления + бонус продвижения
- [ ] Панель предпросмотра боя: показывает БС атакующего/защитника, предсказанный урон до подтверждения
- [ ] Разрешение боя: применяется формула урона; обновляются полосы HP; триггер смерти
- [ ] Анимация боя: 0.5-секундное столкновение спрайтов; возможность пропуска в настройках
- [ ] Система XP: XP начисляется за бой; отслеживание порогов
- [ ] Выбор продвижения: UI карточки бинарного выбора на пороге (2 варианта на класс)

**Защита города**
- [ ] Значение защиты города: вычисляется из стен + гарнизона + местности
- [ ] Дальнобойная атака на города: Осадные юниты снижают защиту за удар
- [ ] Захват города: защита достигает 0 + атакующий рядом → передача владения; население снижается

### Веха 2.3 — 4 играбельные цивилизации (Месяц 4)

- [ ] Экран выбора цивилизации: портрет, цвет, предварительный просмотр способности, тултип начального предпочтения
- [ ] Рим, Египет, Франция, Китай реализованы со всеми уникальными юнитами, зданиями, способностями лидеров
- [ ] UI способности лидера: кнопка с кулдауном и счётчиком ходов, панель описания способности
- [ ] Дифференциация характеристик уникальных юнитов проверена в боевых тест-сценариях
- [ ] Начальное предпочтение: генератор карт уважает предпочтительное размещение местности
- [ ] Основной цвет каждой цивилизации применён к: жетонам юнитов, пограничным тайлам, акцентам UI

### Веха 2.4 — Деревья технологий и инноваций (Месяц 4–5)

**Дерево технологий**
- [ ] Рендерер дерева технологий: горизонтальная прокрутка, разделители эпох, иконки узлов, линии предпосылок
- [ ] Очередь исследований: одно активное исследование; накопление производства → науки
- [ ] Система ускорений: выполнение условий (например, "добыть ресурс") даёт 50% бонус к исследованию
- [ ] Переход эпохи: баннер уведомления + звуковой сигнал эпохи при разблокировке
- [ ] Древняя эпоха завершена (8 технологий)
- [ ] Средние века завершены (7 технологий)

**Дерево инноваций**
- [ ] Рендерер дерева инноваций (та же архитектура, другая цветовая схема)
- [ ] Модальное окно выбора правительства: смена правительства → период золотого века/анархии
- [ ] Выбор карточек политик: выберите 3 из доступного набора; визуальная анимация переворота карты
- [ ] Инновации эпохи Возрождения завершены

**Отложено на Бету:**
- Технологии Возрождения + Индустриальной эпохи (реализованы в Месяце 7)
- Инновации Индустриальной эпохи (реализованы в Месяце 7)

### Веха 2.5 — Базовый ИИ (Месяц 5)

- [ ] Фреймворк дерева поведения интегрирован (кастомный или сторонний)
- [ ] **ИИ сложности «Поселенец»:** Основывает города в допустимых местах; строит очередь производства; исследует технологии; обучает воинов; без боевой агрессии
- [ ] **ИИ сложности «Вождь»:** Все поведения Поселенца + базовое расширение; разведка территории; объявляет войну при достаточной силе; простая торговля ресурсами
- [ ] ИИ не обрабатывается в основном потоке (корутины/задачи для предотвращения просадок FPS)
- [ ] Прохождения ИИ завершают полную игру до победы Науки во внутреннем тестировании
- [ ] Читерство ИИ: нет на Поселенце; незначительные бонусы к выходу на Вожде (видны в дебаге)

### Веха 2.6 — Локальный мультиплеер и базовый UI (Месяц 5–6)

**Локальный (Горячий стул)**
- [ ] Режим горячего стула: игроки чередуются на одном устройстве; экран передачи хода между ходами
- [ ] Отдельные цивилизации выбираемы для каждого слота локального игрока

**Полировка UI (альфа-стандарт)**
- [ ] Все 8 ключевых экранов функциональны (не финальный арт)
- [ ] Встроенное расширение экрана города: слоты районов, очередь строительства, детали населения
- [ ] Дерево технологий/инноваций полностью интерактивно: нажатие на узел ставит в очередь исследование, показывает предпосылки
- [ ] Экран дипломатии: заглушка (показывает только значение отношений + кнопку объявления войны)
- [ ] Экран победы: триггер на условии победы; показывает портрет победившей цивилизации + разбивку счёта
- [ ] Меню настроек: мастер-громкость, громкость SFX, переключатель режима дальтоника, счётчик FPS (дебаг)

**Альфа-плейтестирование**
- [ ] Внутренняя командная сессия: полная игра от начала до победы (все 4 цивилизации, ИИ Поселенца)
- [ ] Сортировка багов: P0/P1 исправлены до ворот Беты; P2/P3 залогированы
- [ ] Проход производительности: FPS профилирован на 3 целевых устройствах; нет устойчивых просадок ниже 25 FPS

**Критерии выхода из Фазы 2:** Полный игровой цикл играбелен внутренне, 4 цивилизации, 2 сложности ИИ, мультиплеер горячего стула, нет P0 крашей.

## Фаза 3 — Бета (Месяцы 7–10)

**Цель:** Полный фичесет, полный контент, онлайн-мультиплеер работает, готово для внешних тестировщиков.

### Веха 3.1 — Оставшиеся цивилизации (Месяц 7)

- [ ] Япония, Аравия, Греция, Россия, Ацтеки, Германия реализованы (все 6 оставшихся)
- [ ] Все 10 лидеров: портретный арт финализирован, 3 эмоциональных состояния (Нейтральное/Дружелюбное/Враждебное)
- [ ] Записаны и интегрированы все 10 голосовых линий лидеров (5 линий на каждого = 50 записей)
- [ ] Проход баланса цивилизаций: проверены данные внутренней игры; применены корректировки характеристик
- [ ] Экран выбора цивилизации: финальный арт, анимированный предпросмотр портрета лидера

### Веха 3.2 — Полные деревья технологий и инноваций (Месяц 7)

- [ ] Технологии эпохи Возрождения (5 узлов) завершены
- [ ] Технологии Индустриальной эпохи (5 узлов) завершены; Освоение космоса → триггер победы Науки
- [ ] Инновации Индустриальной эпохи завершены; разблокирована Демократия
- [ ] Все ускорения технологий (25) реализованы и протестированы
- [ ] Все бонусы инноваций (20) применены и протестированы
- [ ] Полный проход баланса дерева: нет доминирующего оптимального пути; подтверждены несколько жизнеспособных стратегий

### Веха 3.3 — Все условия победы (Месяц 7–8)

- [ ] **Господство:** Определение захвата столицы; триггер победы «последняя столица стоит»
- [ ] **Наука:** Последовательность запуска Космической миссии; многоходовой тип строительства «проект»
- [ ] **Культура:** Трекер туристов; расчёт культурного влияния против других цивилизаций
- [ ] **Религия:** Система основания религии; механики распространения; проверка порога 50% на цивилизацию
- [ ] **Дипломатия:** Трекер сюзеренитета над городами-государствами; система Мирового конгресса (упрощённая); триггер победы 10 ОДП
- [ ] **Очки:** Расчёт очков за ход (население + районы + чудеса + технологии + эпоха); триггер победы по лимиту ходов
- [ ] Уведомление об условии победы: полноэкранный кинематографический показ для победителя + все другие игроки видят результат

### Веха 3.4 — Полная система дипломатии (Месяц 8)

- [ ] Полная шкала отношений: все модификаторы активны и протестированы
- [ ] Дипломатические действия: все 7 действий реализованы с потоками подтверждения
- [ ] Экран торговли: предметы роскоши за роскошь, золото за ход, единовременная сумма золота
- [ ] Научное соглашение: обе стороны инвестируют; бонус применяется при истечении
- [ ] Мирные переговоры: карта передачи территории; денежные требования
- [ ] Система альянсов: взаимный оборонительный пакт; общее видение; совместная война
- [ ] Система городов-государств: 4–6 городов-государств на стандартной карте; механика влияния; бонусы сюзеренитета
- [ ] Дипломатия ИИ: решения на основе отношений; торговые предложения; объявления войны; мирные предложения

### Веха 3.5 — Онлайн-мультиплеер (Месяц 8–9)

**Бэкенд и инфраструктура**
- [ ] Управление сессиями Photon Fusion 2: создание комнаты, вход, выход, повторное подключение
- [ ] Firebase Firestore: документ состояния на матч; оптимистичные обновления + серверная сверка
- [ ] Авторитарная валидация хода: все действия проверяются на стороне сервера перед применением
- [ ] Сжатие дельты состояния: передаются только изменённые поля (целевой показатель ~5 КБ/ход)
- [ ] Тест задержки: круговой путь отправки хода < 300 мс на 4G-соединении

**Система лобби**
- [ ] Создание комнаты: название, приватность (публичная/приватная), размер карты, режим, таймер хода, список банов цивилизаций
- [ ] Приватная комната: 6-значный код обмена + диплинк (открывает приложение прямо в лобби)
- [ ] Публичный быстрый матч: подбор по ELO; таймаут заполнения лобби; добивка ботами
- [ ] Приглашение друзей: интеграция списка друзей платформы (iOS Game Center / Google Play Games)
- [ ] Режим наблюдателя: войти как наблюдатель; задержка 2 хода; чат наблюдателей отдельно

**Режимы ходов**
- [ ] Одновременные ходы: все игроки отправляют действия; сервер разрешает по получении всех
- [ ] Последовательные ходы: индикатор активного игрока; таймер хода (45с / 90с настраивается); авто-пропуск по таймауту
- [ ] Асинхронные ходы: пуш-уведомление FCM в ваш ход; значок бейджа на иконке приложения; таймер 24ч

**Социальные функции**
- [ ] Внутриигровой текстовый чат: история сообщений, фильтр нецензурной лексики, репорт/мут на игрока
- [ ] Колесо эмоций: 6 эмоций, удержание для открытия, отпускание для отправки; анимированное отображение эмоции на столице отправителя
- [ ] Уведомления о дипломатических предложениях: стиль тост-уведомления в игре

**Обработка выходов**
- [ ] Определение отключения: таймаут пинга 30с → передача управления ИИ с уведомлением остальных
- [ ] Повторное подключение: вернувшийся игрок аутентифицируется → получает полное текущее состояние → возобновляет управление
- [ ] Миграция хоста: если хост лобби отключается → автоматически избирается новый хост
- [ ] Голосование за досрочный конец: любой игрок предлагает; требует большинства (>50%) принятия; игра заканчивается победой по очкам
